# Kia & Hyundai Vehicle Theft Analysis

This repository contains data, analysis code, and presentation materials used to examine
trends in Kia and Hyundai vehicle thefts. The purpose of this project is to analyze theft
patterns across cities and time and to clearly communicate findings using reproducible
data analysis.

---

## Project Overview
This project explores:
- Theft patterns across locations and time
- Trends in theft frequency and hotspots
- Summary findings communicated through a presentation deliverable

## How to Run This Project

### 1) Download the Repository
- Click **Code â†’ Download ZIP**
- Extract the folder to your local machine
- Keep the folder structure unchanged

### 2) Review and Download Data Files
- Open the `KIA/Data/` folder
- All CSV files required for the analysis are located here
- Do not rename or move these files, as the code uses relative paths

### 3) Run the R Analysis Script
- Open R or RStudio
- Navigate to `KIA/CodeFiles/`
- Open `Kia_Thefts.R` (or `Kia_Thefts.Rmd`)
- Run the script from top to bottom

The script performs:
- Data loading and cleaning
- Exploratory and trend analysis
- Generation of outputs and visualizations

### 4) Compare Results with the Presentation
- Open the `KIA/Presentation/` folder
- Review the slides/PDF
- Compare outputs from the analysis with the findings presented
