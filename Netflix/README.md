# Netflix Viewership Analysis

## Project Overview
This project analyzes Netflix viewership data to provide clear, actionable insights
for executive decision-makers. Using Power BI, the analysis focuses on identifying
global and regional trends, content popularity, and market performance to support
strategic decisions related to content investment and market expansion.

The objective is to move beyond basic reporting and demonstrate how visual analytics
can support executive-level decision-making in a competitive streaming environment.

---

## Objectives
- Identify global and regional viewership trends
- Highlight top-performing content and markets
- Support executive decision-making through clear data visualization
- Provide data-driven insights to support audience growth and retention

---

## Tools & Technologies
- Power BI (interactive dashboards and visual analytics)
- Microsoft Excel (source datasets and data preparation)

## Key Insights (High-Level)

- Visual analytics enable faster and more effective executive decisions
