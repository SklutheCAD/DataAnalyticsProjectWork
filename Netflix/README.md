# Netflix Viewership Analysis

## Project Overview
This project analyzes Netflix viewership data to provide clear, actionable insights for executive decision-makers. Using Power BI, the analysis focuses on identifying trends, regional performance, and content popularity to support strategic decisions aimed at expanding overall viewership.

The objective is to move beyond basic reporting and deliver insights that can inform content investment, market expansion, and programming strategy at an executive level.

## Objectives
- Identify global and regional viewership trends
- Highlight top-performing content and markets
- Support executive decision-making through clear data visualization
- Provide data-driven insights to help expand and retain Netflix viewership

## Tools & Technologies
- Power BI for interactive dashboards and executive-level visualizations
- Microsoft Excel for source datasets and data preparation

## Folder Structure
Netflix/
- CodeFiles: Supporting files used during analysis
- Data: Raw and processed Netflix viewership datasets
- README.md: Project documentation

## Key Insights (High-Level)
- Viewership patterns vary by region and time period
- Certain content categories consistently drive higher engagement
- Global trends can be used to inform localized content strategies
- Visual analytics enable faster and more effective executive decisions

## Intended Audience
This analysis is designed for executives, senior leadership, strategy teams, and business analytics stakeholders.

## Outcome
By presenting Netflix viewership data through Power BI dashboards, this project demonstrates how analytics can be leveraged to support strategic growth initiatives and improve audience reach in a competitive streaming environment.
